# QueryExpress
学习 HarmonyOS ArkTS 开发时做的小 Demo，使用了[天行数据](https://www.tianapi.com/apiview/152)的快递查询 API。

每日查询量有限，在实际使用时请将相关 key 替换为自己的 API key。

**如果这个项目有帮助到你，拜托请给它点一个 star！**

## Features

- [x] 快递查询
- [x] 已签收未签收快递分类
- [x] 自动保存查询记录
